# 使用指南

QQ 音乐下载器的操作说明。

## 单曲下载

```bash
python song.py
```

1. 选择音质 (`y` = FLAC优先, `n` = MP3优先)
2. 输入歌曲名搜索
3. 选择序号下载

## 歌单下载

```bash
python songlist.py
```

1. 输入歌单链接或 ID
2. 确认下载
3. 选择音质

支持格式：
- ID: `8052190267`
- URL: `https://y.qq.com/n/ryqq/playlist/8052190267`

## 凭证管理

```bash
python credential.py
```

功能：登录 / 检查状态 / 刷新凭证

## 输出文件

- 位置：`./music/` 或 `./music/{歌单名}/`
- 命名：`歌手 - 歌曲.格式`
- 元数据：歌词、封面、专辑信息

## 音质说明

| 模式 | 优先级 |
|------|--------|
| 高品质 (y) | FLAC → MP3_320 → MP3_128 |
| 标准 (n) | MP3_320 → MP3_128 |

👉 配置参数请看 [配置说明](./config)
