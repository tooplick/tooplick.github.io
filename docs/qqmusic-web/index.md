# QQMusic Web

基于 Flask 和 QQ 音乐 API 的在线音乐播放器，支持搜索、播放、下载和音质选择功能。

## 功能特性

- 🎧 **在线播放**：搜索播放歌曲
- 📥 **多音质下载**: 支持标准音质 (MP3) 和无损音质 (FLAC)
- 🐳 **Docker 部署**: 一键部署，开箱即用

## 技术栈

| 类别 | 技术 |
|------|------|
| 后端 | Flask, aiohttp, mutagen |
| 前端 | HTML5, CSS3, JavaScript |
| 异步 | asyncio, aiohttp |
